	Permite que você defina a senha do correio de voz para cada ramal, se for deixado em branco, a senha será a extensão

bulk_extensions.accountcode.tooltip =
	Permite que você defina o código da conta para cada extensão. Você pode usar a palavra reservada <b>&#123;EXTENSÃO&#125;</b>
	para usar a extensão como código de conta.
bulk_extensions.features_password.tooltip =
	Permite que você defina a senha dos recursos para cada extensão, se deixada em branco, uma senha aleatória será gerada.
	Você pode usar a palavra reservada <b>&#123;EXTENSÃO&#125;</b> para usar a extensão como senha de recursos.

	A senha do correio de voz contém caracteres inválidos. Apenas números, símbolo de asterisco ou palavras reservadas são permitidos
bulk_extensions.validation.invalid_f_password =
	A senha dos recursos contém caracteres inválidos. Apenas números, símbolo de asterisco ou palavras reservadas são permitidos

bulk_extensions.name_prefix = Prefixo do nome
bulk_extensions.name_prefix.tooltip =
	Apermite que você defina um prefixo para usar como parte do nome da extensão, por exemplo.: se você definir o valor do prefixo como Agente e
	a extensão é 200, o nome da extensão será <b>Agent 200</b>. Se em branco, a palavra <b>Extensão</b> será usado como prefixo
bulk_extensions.device_password = Senha de dispositivos
bulk_extensions.device_password.tooltip =
	Ele permite que você defina uma senha dinâmica ou fixa para os dispositivos gerados. Se em branco, uma senha aleatória será definida.
	Você pode usar a palavra reservada <b>&#123;EXTENSÃO&#125;</b> para tornar o número de extensão parte da senha,
	por exemplo. : se a extensão for 100 e você definir o seguinte valor para este campo <b>securepwd&#123;EXTENSÃO&#125;</b>, então, a senha será:
	<b>securepwd100</b>
